# React App

간단한 CRUD를 위한 화면

# React App 실행 방법

1. node.js 설치(npm 이용) \
  npm -v 를 이용해 node.js 설치 여부 확인
2. react 설치 \
  npm install -g create-react-app \
  create-react-app -V 를 이용해 react 설치 여부 확인
3. react app 실행할 폴더로 이동 
4. 현재 폴더에 react app 실행 \
   create-react-app .
5. react app 실행 \
   npm run start
